This module enables partial invoicing by percentage directly from each Sale Order Line, allowing you to generate invoices progressively based on project milestones, partial delivery, or phased billing.

  • Create invoices by percentage per sale order line
  • Prevents invoicing more than 100%
  • Tracks remaining percentage automatically
  • Supports multiple partial invoices per line
  • Ideal for services, projects, milestones, retainers, construction billing, etc.

Create Invoice from Sale Order Line

A dedicated Create Invoice button appears on each sale order line, allowing percentage-based invoicing with a single click.

Enter Percentage to Invoice

The wizard automatically shows the remaining percentage and prevents over-invoicing beyond 100%.

Invoice Generated Automatically

The invoice is created for the selected percentage and linked back to the sale order line for accurate tracking.
