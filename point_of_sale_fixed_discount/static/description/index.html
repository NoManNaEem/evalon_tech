POS Fixed Amount Discount (SME)

Apply fixed amount discounts in POS instead of percentage — Odoo 18 Community

This module allows POS users to apply discounts as a fixed amount rather than percentage. It introduces a toggle option in Point of Sale settings to enable or disable fixed discount mode without affecting Odoo’s default discount logic.

Key Features

  • Apply fixed amount discounts on POS order lines
  • Automatically converts percentage values when enabled
  • Discount recalculates correctly when quantity changes
  • POS receipt displays fixed discount value
  • Toggle enable/disable from POS Settings
  • Fully compatible with Odoo 18 Community Edition

How It Works

  1. Go to Settings → Point of Sale
  2. Enable Custom Fixed Discount
  3. Open a POS session
  4. Enter discount as a fixed amount
  5. The system automatically recalculates totals and taxes
