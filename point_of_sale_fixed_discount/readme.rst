POS Fixed Amount Discount — Odoo 18 Community
===================================================

This module allows Point of Sale users to apply discounts as a **fixed amount**
instead of percentage. It introduces a toggle option in POS Settings that enables
fixed discount mode while keeping Odoo’s default discount behavior intact.

Features
--------

- Apply **fixed amount discounts** on POS order lines
- Automatically converts percentage values when enabled
- Recalculates discount correctly when quantity changes
- POS receipt displays fixed discount value
- Toggle enable/disable from **Settings → Point of Sale**
- Fully compatible with **Odoo 18 Community Edition**

Usage
-----

1. Go to **Settings → Point of Sale**
2. Enable **Custom Fixed Discount**
3. Open a POS session
4. Enter discount as a **fixed amount**
5. Totals and taxes are recalculated automatically

Compatibility
-------------

- Odoo **18.0 Community Edition**
- Requires **Point of Sale** module

Installation
------------

1. Install *POS Fixed Amount Discount * from Apps
